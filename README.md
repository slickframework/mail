Slick E-Mail plugin
===================

Email plugin provides generalized functionality to compose and send both text and MIME-compliant multipart email messages.

**Installation**

To use E-Mail plugin in your project just add the following line to your projectâ€™s
`composer.json` file:

    {
        "require": {
            "slick/mail": "*",
            ...
        }
    }

Then you need to run:

    $ composer update
